# HireSynapse - 智能简历解析助手

HireSynapse 是一款强大的 Chrome 浏览器扩展，旨在帮助招聘人员高效地从各大招聘网站（如 LinkedIn、猎聘、BOSS直聘）提取候选人简历信息，并利用 AI 智能解析，最终一键保存到 Notion 数据库中。

## ✨ 核心功能

*   **智能解析**：利用先进的大语言模型 (LLM) 自动提取简历中的关键信息（姓名、联系方式、工作经历、教育背景等）。
*   **多平台支持**：完美支持 LinkedIn、猎聘、BOSS直聘等主流招聘平台。
*   **Notion 集成**：一键将解析后的结构化数据保存到您的 Notion 数据库，支持自定义字段映射。
*   **数据标准化**：内置公司和院校字典，自动标准化名称，确保数据一致性。
*   **双语支持**：全界面支持中文和英文切换。
*   **隐私安全**：API Key 等敏感信息加密存储在本地，不上传任何第三方服务器。

## 🚀 安装指南

1.  **下载源码**：下载本项目的源代码压缩包并解压。
2.  **打开扩展管理**：在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
3.  **开启开发者模式**：点击右上角的"开发者模式"开关。
4.  **加载扩展**：点击左上角的"加载已解压的扩展程序"，选择解压后的项目文件夹。
5.  **完成**：HireSynapse 图标将出现在浏览器工具栏中。

## ⚙️ 配置指南

在使用前，请右键点击插件图标，选择"选项" (Options) 进入配置页面。

### 1. LLM 配置 (AI 模型)
这是解析的核心。您需要配置 OpenAI 兼容的 API。
*   **Base URL**: API 接口地址（例如 `https://api.openai.com/v1` 或您的代理地址）。
*   **API Key**: 您的 API 密钥（将加密存储）。
*   **模型名称**: 配置 Key 后点击"获取模型列表"选择，推荐使用 `gpt-3.5-turbo` 或 `gpt-4`。
*   **System Prompt**: 可自定义提示词，通常保持默认即可。

### 2. Notion 配置 (数据存储)
*   **Integration Token**: 在 [Notion Integrations](https://www.notion.so/my-integrations) 创建一个新的 Integration，获取 Token。
*   **数据库**:
    1. 在 Notion 中创建一个数据库 (Database)。
    2. 点击数据库右上角 `...` -> `Connect to` -> 选择您刚才创建的 Integration。
    3. 在插件设置页点击"获取数据库列表"并选择该数据库。
*   **字段映射**: 选择数据库后，将简历字段（如姓名、电话）映射到您 Notion 数据库的对应列。

### 3. 字典管理 (可选)
为了保持数据整洁，您可以管理公司和院校的标准名称字典。
*   支持导入/导出 JSON 格式的字典文件。
*   支持手动添加、删除条目。
*   解析时，AI 会尝试将提取的名称匹配到字典中的标准名称。

## 📖 使用指南

1.  **打开简历页面**：在浏览器中访问候选人的简历详情页（如 LinkedIn 个人主页）。
2.  **启动插件**：点击浏览器侧边栏图标，选择 HireSynapse（或点击插件图标打开侧边栏）。
3.  **开始解析**：
    *   点击圆形的 **"开始解析"** 按钮。
    *   插件将自动提取页面内容 -> AI 智能分析 -> 字典匹配。
4.  **预览与编辑**：
    *   解析完成后，您可以在侧边栏预览提取到的信息。
    *   如有需要，可以直接修改表单中的内容。
    *   下方会显示生成的 Markdown 格式简历正文。
5.  **保存**：
    *   点击 **"保存到 Notion"**。
    *   成功后，数据将立即同步到您的 Notion 数据库中。
    *   点击 **"继续解析"** 按钮可快速处理下一份简历。

## ❓ 常见问题

**Q: 点击"开始解析"没有反应？**
A: 请确保页面已完全加载。如果问题持续，请尝试刷新页面或重新打开侧边栏。

**Q: Notion 数据库列表为空？**
A: 请确保您已在 Notion 数据库页面将 Integration 添加为连接 (`Connect to`)。

**Q: 解析结果不准确？**
A: 您可以在设置页微调 System Prompt，或者检查页面内容是否完整显示（部分网站需要登录才能查看完整简历）。

**Q: 报错 "TypeError: Cannot set properties of null..."？**
A: 这通常是配置页面的显示问题，请尝试刷新扩展或重新打开设置页。如果持续出现，请联系开发者。
